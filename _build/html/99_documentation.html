Spike Prime Python API#

App module#

The app module is used communicate between hub and app.

Bargraph#

The bargraph module is used make bar graphs in the SPIKE App

To use the bargraph module simply import the module like so:

1from app import bargraph

bargraph.change#

change(color: int, value: float) -> None

  • color: int → A colour from the color module

  • value: float → The value

bargraph.clear_all#

clear_all() -> None

bargraph.get_value#

get_value(color: int) -> Awaitable

  • color: int → A colour from the color module

bargraph.hide#

hide() -> None

bargraph.set_value#

set_value(color: int, value: float) -> None

  • color: int → A colour from the color module

  • value: float → The value

bargraph.show#

show(fullscreen: bool) -> None

  • fullscreen: bool → Show in full screen

Sound#

The sound module is used play sounds in the SPIKE App

To use the sound module simply import the module like so:

1from app import sound

sound.play#

  • play(sound_name: str, volume: int &rarr; 100, pitch: int &rarr; 0, pan: int &rarr; 0) -> Awaitable

Play a sound in the SPIKE App

  • sound_name: str → The sound name as seen in the Word Blocks sound extension

  • volume: int → The volume (0-100)

  • pitch: int → The pitch of the sound

  • pan: int → The pan effect determines which speaker is emitting the sound, with -100 being only the left speaker, 0 being normal, and 100 being only the right speaker.

sound.set_attributes#

set_attributes(volume: int, pitch: int, pan: int) -> None

  • volume: int → The volume (0-100)

  • pitch: int → The pitch of the sound

  • pan: int → The pan effect determines which speaker is emitting the sound, with -100 being only the left speaker, 0 being normal, and 100 being only the right speaker.

sound.stop#

stop() -> None

Colour Matrix#

To use the color_matrix module, add the following import statement to your project:

1import color_matrix

All functions in the module should be called inside the color_matrix module as a prefix like so:

1color_matrix.set_pixel(port.A, 1, 1, color.BLUE, 10)

color_matrix.clear#

clear(port: int) -> None

Turn off all pixels on a Colour Matrix

1from hub import port
2import color_matrix
3
4color_matrix.clear(port.A)

  • port: int → A port from the port submodule in the hub module

color_matrix.get_pixel#

get_pixel(port: int, x: int, y: int) -> tuple[int, int]

Retrieve a specific pixel represented as a tuple containing the colour and intensity

1from hub import port
2import color_matrix
3
4# Print the colour and intensity of the 0,0 pixel on the Colour Matrix connected to port A 
5print(color_matrix.get_pixel(port.A, 0, 0))

  • port: int → A port from the port submodule in the hub module

  • x: int → The X value (0-2)

  • y: int → The Y value (0-2)

color_matrix.set_pixel#

set_pixel(port: int, x: int, y: int, pixel: tuple[color: int, intensity: int]) -> None

Change a single pixel on a Colour Matrix

1from hub import port
2import color
3import color_matrix
4
5# Change the colour of the 0,0 pixel on the Colour Matrix connected to port A 
6color_matrix.set_pixel(port.A, 0, 0, (color.RED, 10))
7
8# Print the colour of the 0,0 pixel on the Colour Matrix connected to port A 
9print(color_matrix.get_pixel(port.A, 0, 0)[0])

  • port: int → A port from the port submodule in the hub module

  • x: int → The X value (0-2)

  • y: int → The Y value (0-2)

  • pixel: tuple[color: int, intensity: int] → Tuple containing colour and intensity, meaning how bright to light up the pixel

color_matrix.show#

show(port: int, pixels: list[tuple[int, int]]) -> None

Change all pixels at once on a Colour Matrix

 1from hub import port
 2import color
 3import color_matrix
 4
 5# Update all pixels on Colour Matrix using the show function 
 6
 7# Create a list with 18 items (colour and intensity pairs) 
 8pixels = [(color.BLUE, 10)] * 9 
 9
10# Update all pixels to show same colour and intensity 
11color_matrix.show(port.A, pixels)

  • port: int → A port from the port submodule in the hub module

  • pixels: list[tuple[int, int]] → A list containing colour and intensity value tuples for all 9 pixels.

Colour Sensor#

The color_sensor module enables you to write code that reacts to specific colours or the intensity of the reflected light.

To use the color_sensor module, add the following import statement to your project:

1import color_sensor

All functions in the module should be called inside the color_sensor module as a prefix like so:

1color_sensor.reflection(port.A)

The Colour Sensor can recognise the following colours:

  • Red

  • Green

  • Blue

  • Magenta

  • Yellow

  • Orange

  • Azure

  • Black

  • White

color_sensor.color#

color(port: int) -> int

Returns the colour value of the detected colour. Use the color module to map the colour value to a specific colour.

1import color_sensor
2from hub import port
3import color
4
5if color_sensor.color(port.A) is color.RED:
6    print("Red detected")

  • port: int → A port from the port submodule in the hub module

color_sensor.reflection#

reflection(port: int) -> int

Retrieves the intensity of the reflected light (0-100%).

  • port: int → A port from the port submodule in the hub module

color_sensor.rgbi#

rgbi(port: int) -> tuple[int, int, int, int]

Retrieves the overall colour intensity and intensity of red, green and blue.

Returns tuple[red: int, green: int, blue: int, intensity: int]

  • port: int → A port from the port submodule in the hub module
